P, Q, R Branch Origin

The vibrational transition $v=0\to1$ is accompanied by simultaneous rotational changes ($\Delta J = 0, \pm 1$):

$$\tilde{\nu}_P = \tilde{\nu}_e - (B_1+B_0)J + (B_1-B_0)J^2$$ $$\tilde{\nu}_R = \tilde{\nu}_e + (B_1+B_0)(J{+}1) + (B_1-B_0)(J{+}1)^2$$

Why No Q Branch for Linear Molecules?

For linear molecules (e.g. HCl, CO), the Q branch is forbidden because there is no rotational angular momentum along the molecular axis, requiring $\Delta J = \pm 1$ only.

Non-linear molecules or those with electronic orbital angular momentum (like NO) allow $\Delta J = 0$, producing a Q branch at $\tilde{\nu}_e$.

Centrifugal Distortion & αe

Because the bond stretches at $v=1$, the rotational constant decreases:

$$B_v = B_e - \alpha_e\left(v + \tfrac{1}{2}\right)$$

This causes P and R branch lines to be unequally spaced — R converges at high $J$ while P opens up.

Temperature Effects

At higher temperatures, higher $J$ levels become more populated (Boltzmann distribution), so lines at larger $J$ grow in intensity and the band broadens.

The most populated level: $J_{\max} \approx \sqrt{k_BT/2hcB_0} - \tfrac{1}{2}$.
