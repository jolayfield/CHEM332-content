Vibrational-Rotational Spectra

Observe realistic P, Q, and R branches with Boltzmann-weighted intensities. Adjust temperature and instrumental resolution to explore the fine structure.

P Branch (ΔJ = −1) Q Branch (ΔJ = 0) R Branch (ΔJ = +1)

P, Q, R Branch Origin

The vibrational transition $v=0\to1$ is accompanied by simultaneous rotational changes. The selection rule $\Delta v = \pm 1$ combines with $\Delta J = 0, \pm 1$:

$$\tilde{\nu}_P = \tilde{\nu}_e - (B_1+B_0)J + (B_1-B_0)J^2$$
$$\tilde{\nu}_Q = \tilde{\nu}_e + (B_1-B_0)J(J+1)$$
$$\tilde{\nu}_R = \tilde{\nu}_e + (B_1+B_0)(J{+}1) + (B_1-B_0)(J{+}1)^2$$

Why No Q Branch for Linear Molecules?

For linear molecules (e.g. HCl, CO), the Q branch is forbidden because there is no rotational angular momentum component along the molecular axis. This requires $\Delta J = \pm 1$, giving only P and R branches.

Non-linear molecules or those with electronic orbital angular momentum (like NO) allow $\Delta J = 0$, producing a Q branch at the band origin $\tilde{\nu}_e$.

Centrifugal Distortion & αe

Because the bond stretches at $v=1$, the rotational constant decreases:

$$B_v = B_e - \alpha_e\left(v + \tfrac{1}{2}\right)$$

This causes the P and R branch lines to be unequally spaced — the R branch converges at high $J$ while the P branch opens up. The asymmetry reveals $\alpha_e$.

Temperature Effects

At higher temperatures, higher $J$ levels become more populated (Boltzmann distribution), so lines at larger $J$ grow in intensity and the band broadens. The most intense line shifts to higher $J$ as $T$ increases.

The most populated level: $J_{\max} \approx \sqrt{k_BT/2hcB_0} - \tfrac{1}{2}$
