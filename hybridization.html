Understanding Orbital Hybridization

Orbital hybridization is a fundamental concept in chemistry that explains molecular geometry and bonding. It describes how atomic orbitals combine (mix) to form new hybrid orbitals.

Mathematical Foundation: Linear Combination of Atomic Orbitals (LCAO)

Hybrid orbitals are formed as normalized linear combinations of atomic orbitals:

$$|\psi_{hybrid}\rangle = c_s|s\rangle + c_{p_x}|p_x\rangle + c_{p_y}|p_y\rangle + c_{p_z}|p_z\rangle$$

where $c_i$ are coefficients that satisfy the normalization condition:

$$c_s^2 + c_{p_x}^2 + c_{p_y}^2 + c_{p_z}^2 = 1$$

sp Hybridization

Geometry: Linear (180°)

Example: C≡C, HCN

Composition:

  • 50% s character
  • 50% p character

$$|\psi_1\rangle = \frac{1}{\sqrt{2}}|s\rangle + \frac{1}{\sqrt{2}}|p_z\rangle$$

sp² Hybridization

Geometry: Trigonal Planar (120°)

Example: C=C, BF₃

Composition:

  • 33% s character
  • 67% p character

$$|\psi_1\rangle = \frac{1}{\sqrt{3}}|s\rangle + \sqrt{\frac{2}{3}}|p_z\rangle$$

sp³ Hybridization

Geometry: Tetrahedral (109.47°)

Example: CH₄, NH₃, H₂O

Composition:

  • 25% s character
  • 75% p character

$$|\psi_1\rangle = \frac{1}{2}(|s\rangle + |p_x\rangle + |p_y\rangle + |p_z\rangle)$$

Key Concepts:

  • Normalization: The sum of squared coefficients equals 1 for proper quantum mechanics
  • Orbital character: Higher s or p character determines orbital shape and properties
  • Geometry prediction: VSEPR theory uses hybridization to predict molecular shapes
  • Bond angles: Determined by the geometry imposed by the hybrid orbital set
  • Bonding strength: Depends on orbital overlap and hybridization type
